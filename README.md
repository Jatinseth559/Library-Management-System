# Library Management System

A Java-based Library Management System built using Object-Oriented Programming concepts to manage books, users, and transactions efficiently.

## Features

- Add, Delete, Update, and Search records
- View profit and analytics
- Display records (in-memory and database)
- Track rental history
- Undo and Redo operations
- User logout functionality

## Tech Stack
- Java
- OOP (Object-Oriented Programming)
- PostgreSQL
- IntelliJ IDEA
